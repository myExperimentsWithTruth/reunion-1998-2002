# PCT Batch 1998-2002 · Silver Jubilee registration

Static registration form for the Pant College of Technology batch of 1998-2002 reunion, hosted on GitHub Pages.

The form collects six fields: name, college ID, branch, mobile, current city, email. Submissions post to a private Google Apps Script web app that validates entries against the batch roster and appends them to a private Google Sheet. No roster data or credentials live in this repository.
